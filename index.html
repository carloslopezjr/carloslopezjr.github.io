HTML Semantic Cheatsheet

Syntax Attribute Description
<header> Header Container usually for either navigational links
or introductory content containing <h1> to <h6> headings.
<nav> Navigation Way to label our navigation links.
<main> Main Used to encapsulate the dominant content within a webpage.
<footer> Footer Contains information that belongs at the bottom of a webpage.
<article> Article Holds content that makes sense on its own. Can hold content such
as articles, blogs, comments, magazines, etc.
<section> Section Defines elements in a document, such as chapters, headings,
or any other area in the document with the same theme.
<aside> Aside Used to mark additional information that can enhance another
element but isn't required in order to understand the main content.
<figure> Figure Element used to encapsulate media such as an image, illustration,
diagram, code snippet, etc.
<figcaption> Figcaption Describes the media in the <figure>. Usually goes inside the <figure> tag.
<audio> Audio Embeds audio content into a document. Uses src to link the audio source.
View attributes.
<video> Video Allows videos to be added to a website. View attributes.
<embed> Embed Can embed any media content including videos, audio files, and gifs from an
external source.View attributes.
